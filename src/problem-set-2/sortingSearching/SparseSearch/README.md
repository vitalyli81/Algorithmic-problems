**Sparse Search**
Given a sorted array of strings which is interspersed with empty strings, write a method to find the location of a given string.

**Examples:**

```
Input : arr[] = ["at", "", "", "", "ball", "", "", "car", "", "", "dad", "", ""]
x = "ball"
Output : 4

Input : arr[] = ["for", "ball", "", "", "", "", "ide", "practice", "", "", "", "quiz"],
x = "ds"
Output : -1
```
